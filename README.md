# Manga Sheet Reset

Google スプレッドシート用 入力初期化ツール  

---

## プライバシーポリシー

本ツールは、ユーザーの個人情報を収集・保存・共有することはありません。
本ツールがアクセスするのは、ユーザー自身が開いているスプレッドシートの範囲のみです。

本ツールは、他のファイルやメール、アカウント情報にはアクセスしません。
